# ABRSM Music Theory Mock Exam App

An Android education app that procedurally generates 29 types of music-theory questions, preparing music students for their ABRSM theory exams.

This app is backed by a backend. The GitHub link can be found here: https://github.com/donaldbeebi/amta-server

# Technical Features

- Jetpack Compose + XML
- core music library module for pitch, chords, time signature etc.
- highly customized user interface for intuitive inputs and question viewing
- canvas-based MusicXML rendering implemented with a custom view
- backed by a RESTful API backend hosted on Google Cloud Functions
